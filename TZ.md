# ТЕХНИЧЕСКОЕ ЗАДАНИЕ  
## Telegram-бот с астрологическими услугами

---

## 1. Описание проекта

Разрабатывается Telegram-бот, предоставляющий пользователям эзотерические сервисы:
- расклады Таро
- нумерологические расчёты и прогнозы
- астрологические гороскопы
- лунные рекомендации

Система использует:
- структурированные базы знаний (справочники, трактовки, методики)
- алгоритмические расчёты (нумерология, базовая астрология)
- AI-слой только для адаптации текста, а не генерации методик

---

## 2. Цели проекта

- Создать Telegram-сервис эзотерических прогнозов
- Обеспечить регулярное использование (retention)
- Реализовать монетизацию (Freemium + подписка + разовые покупки)
- Построить масштабируемую архитектуру
- Минимизировать зависимость от "чистого ИИ" в расчетах

---

## 3. Целевая аудитория

Основной сегмент:
- Женщины 25–45 лет

Дополнительно:
- Женщины 18–25 лет (развлекательный сегмент)
- Мужчины 25–45 лет (финансы/решения)

Интересы:
- эзотерика, Таро, астрология
- психология, саморазвитие
- отношения и жизненные решения

---

## 4. Функциональные требования

### 4.1 Таро
- карта дня
- 1-карточный расклад
- 3-карточный расклад
- расклад на отношения
- расклад «да / нет»
- интерпретации карт (из базы знаний)

### 4.2 Нумерология
- число жизненного пути
- число судьбы
- число личности
- прогноз на день/месяц
- совместимость по датам рождения

### 4.3 Астрология (базовая)
- гороскоп по знаку зодиака
- прогноз на день/неделю/месяц
- любовный и финансовый гороскоп

### 4.4 Лунные сервисы
- фазы Луны
- лунный календарь
- рекомендации по лунному дню

## 5. MVP (первая версия)

### Обязательные функции:
- Telegram-бот (регистрация пользователя)
- карта дня Таро
- расклад 1 карта
- расклад 3 карты

## 12. База знаний

### Источник данных

**Primary**: `sacred-texts.com/tarot/pkt/` — "The Pictorial Key to the Tarot" (A.E. Waite, 1911)

Текст находится в **public domain** (publication date: 1911). Содержит подробные описания всех 78 карт Rider-Waite-Smith.

### Формат данных (JSON)

```json
{
  "id": "major_00_fool",
  "name": "The Fool",
  "name_ru": "Шут",
  "arcana_type": "major",
  "number": 0,
  "suit": null,
  "upright_meaning": "Новые начинания, спонтанность, вера в жизнь...",
  "reversed_meaning": "Безрассудство, страх перемен, застой...",
  "keywords": ["начало", "свобода", "приключение", "наивность"],
  "description": "Карта изображает путника на краю обрыва...",
  "love": "Новые отношения, романтическое приключение...",
  "career": "Новый проект, смена направления...",
  "finance": "Рискованные вложения, начало нового дела..."
}
```

### Язык контента: русский

Все интерпретации хранятся на русском языке. Это упрощает логику, снижает latency, убирает зависимость от переводчика при каждом запросе.

### Процесс сбора

| Шаг | Описание | Время | Инструменты |
|-----|----------|-------|-------------|
| 1 | Парсинг sacred-texts.com | 1 день | Python, BeautifulSoup |
| 2 | Очистка и структурирование | 0.5 дня | regex, json |
| 3 | Перевод на русский | 1 день | deep-translator |
| 4 | Ручная корректировка ключевых терминов | 1 день | Редактор |
| 5 | Дополнение категориями (love/career/finance) | 0.5 дня | Python |
| 6 | Валидация и тестирование | 0.5 дня | pytest |
| **Итого** | | **4.5 дня** | |

## 16. Этапы разработки

### Этап 0: Инфраструктура (3–5 дней)
- Docker-compose: postgres, redis, bot, ollama
- Alembic миграции
- Структура проекта (слойность: handlers → services → repositories)
- Конфигурация (.env, pydantic-settings)
- CI: ruff, mypy, pytest

### Этап 1: База знаний (4–5 дней)
- Парсинг sacred-texts.com (78 карт)
- Структурирование JSON (русский язык)
- Ручная корректировка 50–70 ключевых терминов
- Валидация и тестирование

### Этап 2: Ядро бота (2 недели)
- Регистрация пользователя (Telegram ID + имя + знак зодиака)
- Карта дня (рандом + интерпретация из БЗ)
- Расклад 1 карта, 3 карты
- Число жизненного пути (алгоритм нумерологии)
- Гороскоп на день (по знаку зодиака)
- История запросов (последние 10)

### Этап 3: AI-слой + Луна (1 неделя)
- Ollama + gemma3:4b — адаптация текста под контекст (имя, знак)
- ephem — фазы Луны, лунный календарь, лунные рекомендации
- Кэширование лунных расчётов на месяц вперёд

### Этап 4: Монетизация (1 неделя)
- Лимит freemium (1 запрос/день)
- Telegram Stars интеграция
- Подготовка к ЮKassa (webhook, верификация)
- Premium-подписка (разблокировка)

### Этап 5: Полировка (3–5 дней)
- Админ-панель (CRUD трактовок, просмотр пользователей)
- Логирование, мониторинг, алерты
- Резервное копирование БД
- Тестирование нагрузки

**Общий срок MVP**: ~6–8 недель
